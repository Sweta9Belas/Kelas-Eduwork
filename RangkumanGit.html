Perintah Keterangan
Git Init : Untuk membuat repository pada file lokal yang nantinya ada folder .git
Git Config : Konfigurasi email dan username
Git Status : Digunakan untuk mengetahui status file
Git Add : Digunakan untuk memasukkan file ke staging area
Git Log : Digunakan untuk melihat riwayat commit
Git Commit : Digunakan untuk mencatat perubahan perubahan yang dilakukan
Git Clone : Digunakan untuk mengcopy repo dari server
Git Remote : Semacam Api untuk berinteraksi dari client dengan server
Git Push : Digunakan untuk mengirimkan commit ke server
Git Pull : Digunakan untuk mengambil commit dari server jika ada perubahan di server yang dibuat oleh
branch lain
Git Branch : Digunakan untuk melihat semua branch (lokal)
Git Branch -all: Digunakan untuk melihat semua branch (Public)
Git Branch nama_branch: Digunakan untuk membuat branch baru
Git Branch -d nama_branch: Digunakan untuk menghapus branch
Git checkout nama_branch: Digunakan untuk berpindah branch
Git merge nama_branch: Digunakan untuk menggabungkan commit dari 2 branch
